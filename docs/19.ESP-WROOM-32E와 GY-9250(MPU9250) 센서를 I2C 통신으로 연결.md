### ESP-WROOM-32E와 GY-9250(MPU9250) 센서를 I2C 통신으로 연결하는 방법입니다.

## 기본 I2C 배선
GY-9250 핀      ESP32 핀        설명
VCC             3.3V전원        (3.3V 사용 권장)
GND             GND             접지
SCL             GPIO22          I2C 클럭 (기본 SCL)
SDA             GPIO21          I2C 데이터 (기본 SDA)

## 선택적 핀 연결
GY-9250 핀      ESP32 핀            용도
AD0             GND 또는 3.3V       I2C 주소 선택 (GND=0x68, 3.3V=0x69)
INT             GPIO (예: GPIO4)    인터럽트 핀 (선택사항)
NCS             3.3V                I2C 모드 활성화 (SPI 비활성화)
FSYNC           연결 안 함           보통 사용 안 함

## 주의사항
⚠️ 전압: MPU9250은 3.3V 센서입니다. ESP32의 3.3V 핀에 연결하세요.
⚠️ 풀업 저항: 대부분의 GY-9250 모듈에는 이미 10kΩ 풀업 저항이 내장되어 있습니다.
⚠️ I2C 주소: 기본값은 0x68입니다. AD0 핀을 3.3V에 연결하면 0x69로 변경됩니다.

```
cpp#include <Wire.h>

#define MPU9250_ADDR 0x68
#define SDA_PIN 21
#define SCL_PIN 22

void setup() {
  Serial.begin(115200);
  Wire.begin(SDA_PIN, SCL_PIN);
  
  // MPU9250 확인
  Wire.beginTransmission(MPU9250_ADDR);
  byte error = Wire.endTransmission();
  
  if (error == 0) {
    Serial.println("MPU9250 감지됨!");
  } else {
    Serial.println("MPU9250을 찾을 수 없습니다.");
  }
}

void loop() {
  // 센서 데이터 읽기 코드 추가
  delay(1000);
}
```

다른 GPIO 핀을 사용하고 싶다면 Wire.begin(SDA_PIN, SCL_PIN)에서 핀 번호를 변경하면 됩니다!
